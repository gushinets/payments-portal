# Политика информационной безопасности

## 1. Общие положения

1.1. Настоящая Политика информационной безопасности (далее — «Политика») определяет основные принципы и меры, применяемые Индивидуальным предпринимателем Говоровым Романом Стальевичем (ИНН 143509640374, ОГРНИП 314547633100101) (далее — «Оператор») для обеспечения защиты информации, включая персональные данные пользователей, обрабатываемой в связи с работой сайта https://any-tool-ai.ru и Сервисов Document Summary и Prompt Optimizer.

## 2. Организационные меры защиты

2.1. Оператор назначает ответственных лиц за организацию обработки и защиты персональных данных.

2.2. Доступ к информационным системам, содержащим персональные данные, предоставляется только лицам, которым он необходим для исполнения трудовых или договорных обязанностей.

2.3. Оператор проводит оценку рисков нарушения безопасности персональных данных и принимает меры по их минимизации.

## 3. Технические меры защиты

3.1. Весь обмен данными между пользователем и Сайтом осуществляется по защищённому протоколу HTTPS (TLS-шифрование).

3.2. Сервер, на котором обрабатываются персональные данные пользователей — граждан РФ, физически размещён на территории Российской Федерации.

3.3. Оператор применяет средства защиты от несанкционированного доступа, включая разграничение прав доступа, использование надёжных паролей и, при необходимости, многофакторную аутентификацию для административных учётных записей.

3.4. Регулярно устанавливаются обновления безопасности используемого программного обеспечения и инфраструктуры.

3.5. Оператор не хранит полные реквизиты банковских карт пользователей — обработка платёжных данных осуществляется исключительно платёжным партнёром (агрегатором), сертифицированным по стандарту PCI DSS.

3.6. Данные, необходимые для восстановления работоспособности сервисов, регулярно резервируются.

## 4. Действия при инцидентах безопасности

4.1. При обнаружении инцидента, повлёкшего утечку или неправомерный доступ к персональным данным, Оператор принимает меры по локализации и устранению последствий инцидента.

4.2. В случаях, предусмотренных законодательством РФ, Оператор уведомляет уполномоченный орган по защите прав субъектов персональных данных о произошедшем инциденте в установленные сроки.

## 5. Обязанности пользователя

5.1. Пользователю рекомендуется не передавать третьим лицам данные для доступа к своей учётной записи, использовать надёжный пароль и своевременно сообщать Оператору о подозрительной активности, связанной с его аккаунтом, на info@anytoolai.ru.

## 6. Изменения Политики

6.1. Оператор вправе вносить изменения в настоящую Политику. Актуальная редакция всегда доступна по адресу https://any-tool-ai.ru/ru/security.

## 7. Контакты

По вопросам информационной безопасности обращайтесь: info@anytoolai.ru
