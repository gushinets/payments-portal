Load this guidance before every Payment Portal UI task.

## Identity

Dark premium AI-native interface inspired by Arc, Raycast, and Notion AI.
Use glass surfaces, layered depth, bento layouts, ambient glows, indigo
gradients, and restrained teal live-state accents.

## Core tokens

```text
Base background: #0F0C29 with layered radial glows
Primary accent: linear-gradient(135deg, #6366F1, #818CF8)
Live accent: #5EEAD4
```

## Rules

- Elevated surfaces use translucent backgrounds, borders, and blur.
- Primary buttons use the accent gradient and glow.
- Cards use 16–20px radii; buttons use 8–12px; pills are fully rounded.
- Page backgrounds always include ambient radial glows.
- Desktop content max-width is 1160px.
- Preserve keyboard focus, readable contrast, and responsive single-column
  layouts.
- Never replace the canonical token values with improvised alternatives.
